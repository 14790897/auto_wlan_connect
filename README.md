# 简介
 此app用于快速连接上海师范大学校园网

## 使用方法
1. 连接校园网
2. 关闭流量（因为校内网站禁止校外流量访问，而由于WiFi未连接，所以数据会通过流量走，导致访问被拒绝）
3. 填入用户名密码
4. 点击login
4. 身份验证成功，愉快上网
